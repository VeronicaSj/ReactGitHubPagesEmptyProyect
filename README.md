# React Empty Proyect prepared for GitHub Pages

This is a clean React project configured for GitHub Pages deployment.

## Deploying

1. Install dependencies:
```
npm install
```

2. Push main branch to GitHub:
```
git add .
git commit -m "Initial commit"
git push -u origin main
```

3. Deploy to GitHub Pages:
```
npm run deploy
```

The site will be live at: https://veronicasj.github.io/ReactGitHubPagesEmptyProyect
